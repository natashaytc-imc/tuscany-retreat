# Formspree 設定說明

## 步驟 1：註冊 Formspree 帳號

1. 前往 [Formspree](https://formspree.io/)
2. 點擊「Sign Up」註冊新帳號（可使用 Google 帳號快速註冊）
3. 完成註冊並登入

## 步驟 2：建立新表單

1. 登入後，點擊右上角的「New Form」或「+ New Form」
2. 表單會自動建立，你會看到一個表單 ID（例如：`xvgkqyzw`）
3. 記下這個表單 ID

## 步驟 3：更新網站程式碼

1. 開啟 `index.html` 檔案
2. 找到第 170 行：
   ```html
   <form id="weddingForm" class="wedding-form" action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
   ```
3. 將 `YOUR_FORM_ID` 替換為你的 Formspree 表單 ID
   - 例如：`action="https://formspree.io/f/xvgkqyzw"`

## 步驟 4：設定表單通知（可選）

1. 在 Formspree 儀表板中，點擊你的表單
2. 在「Settings」標籤中：
   - 設定「Email Notifications」：選擇接收通知的電子郵件
   - 設定「Email Subject」：自訂郵件主旨（已設定為「婚禮回覆表單 - 新回覆」）

## 步驟 5：測試表單

1. 開啟你的網站
2. 填寫表單並提交
3. 檢查：
   - 網站上是否顯示成功訊息
   - Formspree 儀表板中是否出現新回覆
   - 你的電子郵件是否收到通知（如果已設定）

## 查看回覆

### 在 Formspree 儀表板查看：

1. 登入 [Formspree](https://formspree.io/)
2. 點擊你的表單
3. 在「Submissions」標籤中查看所有回覆
4. 可以：
   - 查看每筆回覆的詳細內容
   - 匯出為 CSV 格式
   - 搜尋和篩選回覆

### 透過電子郵件查看：

- 如果已設定電子郵件通知，每次有新回覆時會收到郵件
- 郵件會包含所有表單欄位的內容

## 表單欄位說明

表單會自動收集以下資訊：
- **姓名**：必填
- **電子郵件**：必填
- **是否攜伴**：格式化為「是，我會攜伴」或「否，我一個人參加」
- **參加人數**：如果選擇攜伴
- **是否同住 Villa**：格式化為「是，我會與你們同住」或「否，我會自行安排住宿」
- **飲食需求**：多選項會以「、」分隔（例如：「素食、過敏」）
- **特殊需求或備註**：文字輸入
- **Day 2 自費活動**：多選項會以「、」分隔
- **祝福留言**：文字輸入
- **提交時間**：自動記錄提交時間（台灣時區）

## 免費方案限制

Formspree 免費方案：
- 每月 50 筆提交
- 基本功能
- 電子郵件通知

如果需要更多功能，可以升級到付費方案。

## 疑難排解

### 如果表單提交後沒有收到回覆：

1. 檢查表單 ID 是否正確
2. 檢查 Formspree 儀表板中是否有錯誤訊息
3. 檢查瀏覽器開發者工具（F12）> Console 標籤，查看是否有錯誤
4. 確認表單欄位名稱是否正確

### 如果需要修改表單欄位：

- 修改 `index.html` 中的表單欄位
- 確保每個輸入欄位都有 `name` 屬性
- Formspree 會自動接收所有有 `name` 屬性的欄位
